Information about CSS

CSS stands for Cascading Style Sheet.CSS is the language We use to style an HTML document.CSS describes how HTML element should be displayed.CSS can be used for very basic document text styling like changing color and size of headings and links.It can be used to creat a layout.It can even be used for effects such as animation.CSS is rule based language you define the rules by specifying groups of styles that should be applied to particular alements or groups of elements on your web page.CSS properties have different allowable values,depending on which property is being specified.There are three different ways to apply CSS to HTML documnet 1.Inline css- Inline css contains the property in the body section attached to the element is known as inline CSS.THis kind of style is specified within an HTML tag using the style attribute. 2.External css-External CSS contains separate css files that contains style properties with the help of tag attributes.CSS property is written in a seprate file with a .css extension and should be linked to the HTML document using a link tag. 3.Embedded or Internal css-This can be used when a single HTML document must be style uniquely.The css rule set should be within the HTML file in the head section.
